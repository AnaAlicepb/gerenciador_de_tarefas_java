Sobre o projeto

O "Gerenciador de Tarefas" é uma aplicação em Java que permite ao usuário gerenciar uma lista de tarefas. O usuário pode adicionar, listar e remover tarefas através de um menu interativo exibido no terminal.

Parte Técnica

  • Linguagem: Java
  • IDE: Visual Studio Code
  • Funcionalidades:
    • Adicionar tarefas com validação para evitar duplicação.
    • Listar todas as tarefas cadastradas.
    • Remover tarefas por índice.
    • Verificação de entrada para garantir a integridade dos dados inseridos.

Estrutura dos Arquivos

  • GerenciadorTarefas.java: Contém a implementação do gerenciador de tarefas, incluindo os métodos para adicionar, listar e remover tarefas.

Casos de Uso

  • Listar Tarefas: Exibe a lista de tarefas cadastradas.
  • Adicionar Tarefa: Adiciona uma nova tarefa à lista, verificando se já existe para evitar duplicações.
  • Remover Tarefa: Remove uma tarefa pelo índice fornecido pelo usuário.
  • Sair: Encerra a execução do programa.
